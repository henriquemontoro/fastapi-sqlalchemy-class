# Remocao de Livro

## Descricao
Remove permanentemente um livro da base de dados com base no ID informado.

## Parametros
- **book_id** (int): ID unico do livro a ser removido. Exemplo: 1

## Resposta
- **200 OK**: Retorna uma mensagem confirmando a remocao.
- **404 Not Found**: Retorna um erro se o livro nao for encontrado.
