# Remocao de Autor

## Descricao
Remove permanentemente um autor da base de dados com base no ID informado.

## Parametros
- **author_id** (int): ID unico do autor a ser removido. Exemplo: 1

## Resposta
- **200 OK**: Retorna uma mensagem confirmando a remocao.
- **404 Not Found**: Retorna um erro se o autor nao for encontrado.
